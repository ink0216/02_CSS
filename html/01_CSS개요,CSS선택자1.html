CSS(Cascading Style Sheets, 종속(포함) )

    CSS = 마크업 언어(html)가 실제로 화면에 표시되는 방법을 기술하는 언어    
        밋밋한 html에 시트지로 꾸미기
        (모양, 색상, 크기, (배경)이미지, 움직임, ...)
        하나의 요소에 여러 CSS코드를 겹쳐서 원하는 스타일을 만들어냄
        W3C=웹 표준 지정 단체
        W3C에서 CSS를 웹 표준으로 지정함

CSS 선택자(CSS Selector)

        현재 문서 내에서 특정 요소를 선택하는 문자 또는 문자열(기호)
        
        
:시작태그 ///  <>안에 작성하는 style 등은 속성값
            두 태그 사이는 내용
        
 : 종료태그
        시작태그와 종료태그까지의 전체가 하나의 요소!

        *요소(Element) : 시작태그 + 속성 + 내용 + 종료태그
    ** CSS 코드를 작성하는 방법 **
  • 방법1 : head 태그에 style 태그를 작성한 후 내부에 css코드를 작성한다(이걸 할거임)
  • 방법2 : 원하는 요소에 직접 style 속성을 이용해서 css코드를 작성한다
  • 방법3 : 외부에 .css 파일을 생성하여 연결한다

태그 선택자

        -현재 문서 내에 특정 태그를 모두 찾아 선택하는 선택자
        h1 태그들을 찾아서 그 색만 빨간색으로 바꾸기
        
        [작성법]
        
        태그명 {css코드;}

id(#) 선택자

CSS, javascript에서 id를 #으로 나타냄 
        HTML 문서 내에서 id가 일치하는 요소를 찾아내는 선택자이다
        [작성법]
        
        
        #id 속성값{ CSS 코드; }

class(.) 선택자

        HTML 문서 내에서 
        class 속성 값이 일치하는 모든 요소를 
        선택하는 선택자
        
        [작성법]
        
            .class속성값{ css 코드;} 
        
        ***요소에 CSS를 적용하기 위해 선택자를 작성할 때
        id 선택자보다 class선택자를 이용하는 것이 좋다!! *** 
        (id는 하나밖에 못해서 범용성 떨어지고,
        id 속성 자체가 CSS에서 보다 JS에서 많이 사용되기 때문!)
